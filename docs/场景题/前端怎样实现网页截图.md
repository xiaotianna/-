# 前端怎样实现网页截图

- 飞书文档，内容在列表页想要查看
- 内容导出为png
- 设计类软件，出图

以内容导出为png为例：
- 账单
- 文档导出png

## 方案

- 截图
    - canvas
    - puppeteer（无头浏览器）
    - html2canvas（原理是canvas）
- 上传cdn
